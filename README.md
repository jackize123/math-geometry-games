# 幾何互動教室

國小高年級數學「面積」單元的兩款互動教學遊戲，純靜態網頁，無需後端。

## 內容

| 遊戲 | 路徑 | 說明 |
| --- | --- | --- |
| 找高大作戰 | `find-height/` | 給定底邊，找出相對應的高。涵蓋平行四邊形、三角形（含鈍角三角形外部高）、梯形。 |
| 複合圖形面積挑戰 | `composite-area/` | 將複合圖形拆解成基本圖形，分別計算面積後以加法或減法合併求總面積。 |

兩款遊戲各有三個關卡，答題後即時回饋並依答錯次數評定星等。

## 技術說明

- 純 HTML / CSS / 原生 JavaScript，無建置流程、無框架、無相依套件。
- 圖形以 SVG 動態繪製。
- 外部資源僅 Google Fonts 與 FontAwesome（皆由 CDN 載入）。

## 本機預覽

因為遊戲以相對路徑載入 `style.css` 與 `game.js`，直接以 `file://` 開啟通常可行，但建議起一個本機伺服器以貼近實際部署環境：

```bash
npx serve .
```

## 部署

本專案為靜態網站，儲存庫根目錄即網站根目錄，不需要任何建置指令或輸出目錄設定。
